sudah beli tiket belum  = nunga tuhor karsis dang
aku pesan tiket bus = au tona karsis motor panumpang
kamu mau pesan tiket apa untuk mudik = ho ra tona karsis aha di mandege huta
jadi rencana saya mau pesan tiket lebih awal = saut tahi au ra tona karsis lobi mula
membelikan tiket = manuhorhon karsis
tiket keberangkatan = karsis haborhaton
nanti tidak dapat tiket hari senin = annon ndada dohar karsis ari senen
tidak ada duit buat beli tiket pak  = ndada isi hepeng bahen tuhor karsis amang
bagaimana aku bisa pulang, tidak ada tiket kereta api = beha au boi mulak ndada isi karsis kareta api
saya mau beli tiket pesawat di mana  = au ra tuhor karsis hopal di dia
